<h3 align="center">Sample libraries for <a href="https://sanicboom.com">Sanic Boom</a></h3>

<p align="center">
  154 free, open-source sample libraries for live coding music.<br/>
  Import any library with a single line:
</p>

```sanic
import "github:SanicBoom/salamander-grand-piano"

let piano = sample("lib:salamander-grand-piano")

let chords = [C4 E4 G4 C5]
    |> piano
    |> reverb(size: 0.6, mix: 0.3)
```
